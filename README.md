# 🏀 Basketball Scoreboard App

A simple basketball scoreboard built using **HTML, CSS, and JavaScript**. This project allows users to track and 
update scores for two teams: Home and Guest.

---

## 🚀 Features

- Add +1, +2, or +3 points for each team
- Real-time score updates using JavaScript DOM manipulation
- Simple and responsive UI

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript

---

## 💡 What I Learned

- DOM manipulation using `getElementById`
- Handling button click events with `onclick`
- Updating UI dynamically with JavaScript
- Basic state management using variables
